Documentation Overview

This documentation describes a simple API that has two endpoints. One allows the user to create a message that has a name, email, and unique token attached. The other allows the user to use the unique token to access the originally attached message, name, and email.

When a message is created, a unique token is generated. Tokens automatically expire after 24-hours or after they have been used.
